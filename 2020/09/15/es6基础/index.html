es6基础

let

  • 局部变量，会在在{}内有意义

  • 变量不可重复定义

  • 不会变量提升

  • 不影响作用域链

const

  • 和let类似
  • 声明的时候必须赋值
  • 赋值后就不可再进行赋值
  • 一般常亮进行大写，约定俗成
  • 定义的数组和对象，里面的值是可以进行修改的

解构赋值

ES6允许按照一定模式从数组和对象中提取值，对变量进行赋值

模版字符串

ES6中引入的新的字符串方式，反引号``

  • 直接出现换行符
  • 变量拼接

对象简化写法

箭头函数

  • this时静态的，始终指向函数声明时所在作用域下的下的this的值
  • 不能作为构造函数来实例化对象
  • 不能使用argment变量
  • 箭头函数不适合作为this有关的回调，事件回调，对象的方法
  • 简写
    • 当只有一个参数时，省略小括弧 b=>{}
    • 函数执行内容只有一行时，可以省略大括号，默认添加return , 比如： b=>b

函数默认值

  • 约定俗成，一般具有默认值的参数放后面
  • 与解构相结合

rest参数

  • 函数中，rest参数必须指定为…args，必须放所有参数的后面
  • args对象返回的是一个数组

扩展运算法

  • 语法：…，三个点号的形式
  • 将数组转换为以逗号分隔的参数序列

应用

  • 数组合并
  • 数组克隆
  • 将伪数组转换为真正的数组

Symbol

  • JavaScript第七种数据类型，表示独一无二的值，是一种类似字符串的数据类型
  • Symbol的值是唯一的，用来解决命名冲突的问题
  • Symbol值不能与其他数据进行运算
  • Symbol定义的对象属性不能使用for..in循环遍历，但是可以使用Reflect.ownKeys来获取对象的所有健名

迭代器

  • Iterator是一种接口，为各种不同的数据结构提供统一的访问机制。任何数据结构只有部署了Iterator接口，就可以完成遍历操作
  • ES6创造了一种新的遍历命令for…of循环
  • 原生具备Iterator接口的数据结构有
    • Array
    • Argument
    • Set
    • Map
    • String
    • TypedArray
    • NodeList
  • 原理：通过指针对象不断next遍历即可

自定义结构迭代

生成器函数

  • 异步编程

  • 语法

    function * name(){}

    在function和函数名字之间加*来表示生成器函数

  • 函数返回一个iterator对象，通过next执行

集合Set

语法

  • new Set()；空集合
  • new Set([])；接收一个可迭代数据，可以用for of迭代
  • 集合内的数据不会重复，会自动去重

Map

  • 键值对
  • 升级版本的对象
  • 键可以是任意类型的数据
  • 值也是可以任意类型的数据

class

  • constructor构造函数，可以在构造函数中添加属性
  • 普通成员函数
  • 静态类函数 static，不是实例对象，和java等不一样，不可混用

继承

对象方法扩展

  • Object.is 判断两个对象是否相等
  • Object.assign 两对象合并
  • Object.setPrototypeOf 设置对象原型
  • Object.getPrototypeOf 获取对象原型

模块化

  • ES6采用的是export和import方式
