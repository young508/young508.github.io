简介

hexo

  • hexo是一款基于node.js的静态博客框架，依赖少，易于安装使用。
  • 功能丰富，操作简单，主题可以随意配置，命令式生成，部署
  • 完美的支持blog最常用的md格式文档

github

  • 全球最大的开源代码提交库，免费+功能强大
  • hexo的代码可以通过托管到github的方式来实现个人blog网站
